🐢 4) مشكلة debounce في search (كويس لكن ناقص تحسين)
عندك:

❌ المشكلة:
• مفيش reuse للـ debounce
• search مختلفين ممكن يحتاجوا نفس system
✅ الأفضل:
js
function debounce(fn, delay = 300) {
  let t;
  return (...args) => {
    clearTimeout(t);
    t = setTimeout(() => fn(...args), delay);
  };
}
